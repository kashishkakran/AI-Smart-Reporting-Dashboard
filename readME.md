# AI-Powered Smart Reporting Dashboard

**Description:**  
This is a Streamlit web app that automates customer success and solutions consulting reporting. Users input customer/project details, and the app generates structured AI-powered reports including Summary, Next Steps, Risks, and Success Metrics. The app also tracks metrics like Time Saved and Report Quality Score, and stores reports in a SQLite database.

## Features

- Input form for structured customer/project data
- AI-generated professional reports (using OpenAI GPT-4)
- Metrics dashboard showing time saved and report quality
- Stores all reports in SQLite database
- Download reports as TXT files
- View previous reports

## Tech Stack

- Python
- Streamlit
- OpenAI API
- SQLite
